1) create EC2 instance

2) open below ports
        8000. (uvicorn)  TCP
        8080. (jenkins). TCP
        80. (Docker).  HTTP

3) connect to EC2 and update ubuntu

    sudo apt update -y

4) install Jenkins

        jenkins.io
        https://www.jenkins.io/doc/book/installing/linux/

5) install Docker
        sudo apt update -y
        sudo apt install docker.io -y
        docker --version

6) enable permissions
        docker jenkins
        docker ubuntu
        jenkins ubuntu

        sudo usermod -aG docker jenkins
        sudo usermod -aG docker ubuntu
        sudo usermod -aG jenkins ubuntu

        sudo systemctl restart jenkins
        sudo systemctl status jenkins
